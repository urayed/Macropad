# Macropad
This is Macropad version 1.

The Macropad has: 
1. 9 MX Keys
2. 1 OLED Screen
3. 1 Rotary Encoder
4. 1 3x3 LED Matrix

As of now, the firmware is likely very bare bones. For the future, I plan to create custom software for programming the keyboard without code. I also want to be able to control the OLED screen and LED Matrix for future projects, so that is also a project, itself, to work on.
